# card-slider

A cart slide to shuffle cards by clicking prev, next and random buttons.

Languages used: JavaScript, HTML, CSS

Methods used:

- Array
- reusable function()
- img.src
- window.addEventListener('DOMContentLoaded', function(){...})
- Math.floor(Math.random())

Steps:

- Create an array
- Select the items
- Set the initial number
- Create a reusable function to show person
- Add an event to load the first person right after window loaded
- Add click event to next, prev and reset buttons
